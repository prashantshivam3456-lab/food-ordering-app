# 🍔 Food Ordering Web App

A modern and user-friendly **Food Ordering Web Application** that allows users to browse food items, add them to cart, and simulate an online ordering experience. This project demonstrates both **frontend design** and **basic backend integration**.

---

## 🚀 Features

✨ Clean and responsive UI
🛒 Add to Cart functionality
📜 Dynamic food menu display
⚡ Smooth user interaction using JavaScript
🌐 Basic backend setup using Node.js
📦 Organized project structure

---

## 🛠️ Tech Stack

**Frontend:**

* 🌐 HTML
* 🎨 CSS
* ⚡ JavaScript

**Backend:**

* 🖥️ Node.js (`server.js`)

---

## 📂 Project Structure

```
📁 Project Folder
 ├── index.html
 ├── style.css
 ├── script.js
 ├── server.js
 ├── package.json
```

---

## 💡 How It Works

1. User opens the website
2. Food items are displayed on the homepage
3. User can add items to cart 🛒
4. JavaScript handles dynamic interactions
5. Backend (Node.js) manages server-side logic
